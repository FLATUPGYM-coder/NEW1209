1. **「安心」を視覚化する**

3. **「変化」を色彩と形で伝える**

### 「ストーリー型・段階的開示」アプローチ

従来のグリッドレイアウトではなく、**訪問者の心理的な旅**に沿ったレイアウト：

- スクロール時に要素が段階的に現れる

## 5. シグネチャー要素（3つの視覚的モチーフ）

1. **「Glowing Accent Line」**
   - 各セクションの見出しの下に、ピンク→ブルーのグラデーション線
   - 動きのあるアニメーション（スクロール時に左から右へ伸びる）
   - FLATUPGYMの「活力」を象徴

2. **「Floating Cards」**
   - 料金プラン、お客様の声をカード形式で表示
   - ホバー時に浮き上がる（shadow + translateY）
   - 3D的な奥行き感を演出

3. **「Circular Motion Elements」**
   - 背景に配置された円形のグラデーション要素
   - ゆっくり回転するアニメーション
   - 「変化」と「継続」を暗示

## 6. インタラクション哲学

### ホバー・クリック時の反応

- **ボタン**: 背景色の変化 + 軽いスケール（1.05倍）+ 影の強化
- **カード**: 浮き上がり効果 + 背景の微かなグラデーション変化
- **テキストリンク**: アンダーラインのアニメーション出現

### スクロール時の反応

- **フェードイン**: 要素が画面に入ると、下からフェードインして登場
- **パラレックス**: 背景画像が遅れてスクロール（軽微）
- **段階的表示**: 複数の要素が時間差で現れる

### 固定フッターボタン

- 常に画面下部に固定
- ホバー時に各ボタンが拡大 + 色が変わる
- クリック時に軽いフィードバック（ripple effect）

## 7. アニメーション指針

### Framer Motion活用

| アニメーション | トリガー | 効果 | 目的 |
| :--- | :--- | :--- | :--- |
| **Fade In + Slide Up** | スクロール進入時 | 下から上へ、透明度0→1 | 要素の登場感を演出 |
| **Glow Pulse** | 見出しテキスト | 光が脈動 | 注目を集める |
| **Float** | カード全体 | 上下に浮遊 | 生命感を表現 |
| **Rotate** | 背景円形要素 | ゆっくり回転 | 継続的な活力 |
| **Bounce** | CTAボタン | 軽いバウンス | 親近感、遊び心 |
| **Ripple** | ボタンクリック時 | 波紋が広がる | フィードバック感 |

### 速度・タイミング

- **標準**: 300-500ms（ホバー、軽いアニメーション）
- **ゆっくり**: 800-1200ms（スクロール進入、重要な要素）
- **背景**: 3000-5000ms（背景要素、常時アニメーション）

## 8. タイポグラフィシステム

### フォント選定

| 用途 | フォント | 理由 |
| :--- | :--- | :--- |
| **見出し（H1-H3）** | `Poppins Bold / ExtraBold` | モダン、親近感、女性らしさ |
| **本文** | `Noto Sans JP Regular` | 読みやすさ、日本語対応 |
| **強調テキスト** | `Poppins SemiBold` | メリハリ、視線誘導 |
| **小見出し** | `Poppins Medium` | 階層感 |

### 階層構造

- **H1**: 48px / Bold / ピンク + グラデーション
- **H2**: 36px / Bold / ネイビー
- **H3**: 24px / SemiBold / ピンク
- **Body**: 16px / Regular / ネイビー
- **Small**: 14px / Regular / グレー

### 行間・文字間隔

- **見出し**: 行間 1.2
- **本文**: 行間 1.6
- **強調**: 文字間隔 +0.5px
